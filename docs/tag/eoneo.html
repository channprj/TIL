조건문

다른 언어들과 마찬가지로 if, elif, else 조건문이 있음. 여러줄일 경우 들여쓰기로 블록을 지정해야 함. 예시는 아래와 같음.

if 문

    if a == 1:
        print("a is 1 !!")
    elif a == 2:
        print("a is 2 !!")
    else:
        print("a is not 1 and 2")
