SVG (Scalable Vector Graphics)

2D 그래픽을 표현하기 위해 XML로 정의된 마크업 언어.

  • Vector Shape
  • Images
  • Text

Group, Style, Transform 가능.

  • Embedding
  • Scripting

SVG의 선, 도형 하나한가 DOM 엘리먼트로 동작하여 이벤트를 걸 수 있음.

SVG(VML) vs Canvas

DOM은 구조적으로 복잡하여 로딩 속도가 느려질 수 밖에 없음. Canvas는 단일 엘리먼트라 상대적으로 속도가 빠를 수 밖에 없음.
